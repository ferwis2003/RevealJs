1. Las plataformas activas actualmente para las que vale la pena desarrollar.

1- Back4App

Back4App es una plataforma de backend de aplicaciones móviles y web que proporciona a los desarrolladores una base de datos flexible y automatizada, alojamiento instantáneo y otras características de backend.

2- FireBase

Firebase es una plataforma de desarrollo de aplicaciones para dispositivos móviles y web desarrollada por Firebase, Inc. en 2011, luego adquirida por Google en 2014.

3- Appy Pie

Appy Pie es una de las mejores plataformas de desarrollo de aplicaciones sin código que tiene su sede en Virginia, Estados Unidos.

3. Tipos de aplicaciones (nativas, híbridas, etc)
  • Nativas
  • Hibridas

* Aplicaciones Nativas

Las aplicaciones nativas (Native App) son aquellas en las que se lleva a cabo específicamente un desarrollo para cada una de las plataformas entonces si quisieramos tener nuestra aplicacion para IOS y Android tendriamos que hacer dos desarrollos aparte.

Los lenguajes que se utilizan para hacer aplicaciones nativas, son Java y Kotlin por la parte de Android y Swift y Objetive-C por la parte de IOS

* Aplicaciones Hibridas

Las aplicaciones híbridas (Hibrid App) o Multiplataformas (Cross Platform) son aquellas que nos permiten desarrollar para varias plataformas a la vez con tan solo realizar un desarrollo.

Los lenguajes que mas se utilizan para esto son Dart y JavaScript

5. Frameworks que se pueden usar para desarrollar aplicaciones (Menciones 3 y cuál es su version actual).

  • Flutter:

    Es un framework de código abierto desarrollado por Google para la construcción de aplicaciones móviles y web. Utiliza el lenguaje de programación Dart y proporciona un conjunto de widgets personalizables.
    La versión actual es Flutter 2.2

  • React Native:

    Es un framework de código abierto creado por Facebook. Se utiliza para desarrollar aplicaciones para Android, iOS, Web y UWP.
    La versión actual es React Native 0.65

  • Ionic

    es un framework de código abierto para la construcción de aplicaciones móviles híbridas utilizando tecnologías web como HTML, CSS y JavaScript. Permite la creación de aplicaciones para iOS, Android y la web.
    La versión actual es Ionic 5.6.11
